# C++ API Reference

The C++ API reference is generated from the upstream `epiworld` source code
using Doxygen plus `mkdoxy`.

## What This Section Is For

- Inspect class signatures and inheritance.
- Read source-level comments for models, tools, viruses, and utilities.
- Jump from the guide into the exact class or file that implements a concept.

## What This Section Is Not For

- It is not the primary onboarding path for new users.
- It should not duplicate the narrative explanations from the guide.
- It should not become the only place where high-level examples live.

## Recommended Use

Start in the [Guide](../guide/index.md), move to [Examples](../examples/index.md)
when you want runnable code, and use the generated API pages when you need the
low-level C++ details.
