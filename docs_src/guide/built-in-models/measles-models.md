## R Note

The current `epiworldR` site notes that measles-specific models now live in the
separate `measles` R package, so this family should be documented primarily
through the C++ reference and disease-specific narrative pages unless you add
explicit material for that R package.

## Choose This Family When

- You need measles-specific states such as prodromal and rash periods.
- Contact tracing, quarantine, or school/entity logic are first-class features
  of the analysis.
- Generic SEIR models are no longer expressive enough for the scenario.
