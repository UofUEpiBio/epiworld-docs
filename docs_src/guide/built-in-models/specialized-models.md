This family groups the models that are conceptually distinct from the standard
compartmental templates.

## Models In This Family

| Model | Description | API |
| --- | --- | --- |
| SIR Logit | Transmission and recovery depend on agent features. | [classModelSIRLogit](../../epiworld/classModelSIRLogit.md) |
| Diffusion Network | Network diffusion rather than standard epidemic compartments. | [classModelDiffNet](../../epiworld/classModelDiffNet.md) |
| Surveillance | Surveillance-oriented model. | [classModelSURV](../../epiworld/classModelSURV.md) |

Shared mechanics are documented once in [Common Patterns](common-patterns.md).

## Minimal Example

These models diverge the most from the standard compartmental templates, so the
family-specific API page should be treated as the authoritative constructor
reference.

=== "C++"

    ```cpp
    epimodels::ModelSIRLogit<> model(/* family-specific arguments */);
    ```

=== "R"

    ```r
    library(epiworldR)

    n <- 100000
    X <- cbind(
      Intercept = 1,
      Female = sample.int(2, n, replace = TRUE) - 1
    )

    model_logit <- ModelSIRLogit(
      "covid2",
      data = X,
      coefs_infect = c(0.1, -2, 2),
      coefs_recover = rnorm(2),
      coef_infect_cols = 1L:ncol(X),
      coef_recover_cols = 1L:ncol(X),
      prob_infection = 0.8,
      recovery_rate = 0.3,
      prevalence = 0.01
    )
    ```

## Choose This Family When

- Agent features directly affect transmission or recovery.
- The research question is closer to diffusion or surveillance than to standard
  compartmental dynamics.
- You need a built-in starting point for a more specialized extension.
