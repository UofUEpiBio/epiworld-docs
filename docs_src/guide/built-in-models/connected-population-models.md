## Minimal Example

The exact constructor signature depends on the connected model you choose, so
the API pages above are the source of truth for arguments. The overall usage
pattern stays simple:

=== "C++"

    ```cpp
    epimodels::ModelSEIRCONN<> model(/* see API reference for arguments */);
    model.run(100, 2587);
    ```

=== "R"

    ```r
    library(epiworldR)

    model <- ModelSEIRCONN(
      name = "COVID-19",
      prevalence = 0.01,
      n = 10000,
      contact_rate = 10,
      incubation_days = 7,
      transmission_rate = 0.10,
      recovery_rate = 1 / 7
    )

    run(model, ndays = 100, seed = 2587)
    ```

## Choose This Family When

- The population is well mixed enough that an explicit network is unnecessary.
- You want faster setup with fewer network design choices.
- You want a clean bridge between compartmental intuition and agent-based
  execution.
